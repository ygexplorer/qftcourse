# 第4章：规范场的量子化

> **前置知识**：[[第2章：路径积分量子化]], [[第3章：规范场基础]]

!!! abstract "本章概要"
    本章是量子场论的技术核心。由于规范场存在"冗余自由度"，直接量子化会导致传播子不可逆（发散）。我们必须使用 Faddeev-Popov 技巧。

---

## 4.1 量子化的难点

### 传播子的困境

规范场的传播子在动量空间的形式：

$$D_{\mu\nu}(p) = \frac{-i}{p^2 + i\epsilon} \left( g_{\mu\nu} - (1-\xi) \frac{p_\mu p_\nu}{p^2} \right)$$

问题在于：矩阵 $P_{\mu\nu} = g_{\mu\nu} - \frac{p_\mu p_\nu}{p^2}$ 的行列式为 0（不可逆）。

!!! warning "物理含义"
    这反映了规范场的冗余自由度——纵分量和标量分量不是独立自由度。

---

## 4.2 阿贝尔理论（QED）

直接在拉格朗日量中加入规范固定项：

$$\mathcal{L}_{\text{gf}} = -\frac{1}{2\xi} (\partial_\mu A^\mu)^2$$

光子传播子变为：

$$D_{\mu\nu}(p) = \frac{-i}{p^2 + i\epsilon} \left( g_{\mu\nu} - \xi \frac{p_\mu p_\nu}{p^2} \right)$$

常用的规范：
- **Feynman 规范**：$\xi = 1$
- **Landau 规范**：$\xi = 0$

---

## 4.3 法捷耶夫-Popov 方法

### 核心思想

1. 提取规范轨道的体积（这体积是无限的）
2. 将其表示为 Faddeev-Popov 行列式的泛函积分
3. 引入鬼场来表示该行列式

### 具体步骤

对于非阿贝尔规范场，拉格朗日量：

$$\mathcal{L} = \mathcal{L}_{\text{YM}} + \mathcal{L}_{\text{gf}} + \mathcal{L}_{\text{FP}}$$

其中鬼场拉格朗日量：

$$\mathcal{L}_{\text{FP}} = -(\partial_\mu \bar{c}^a)(\partial^\mu c^a - gf^{abc}A_\mu^b c^c)$$

!!! success "关键洞察"
    鬼场是**费米性的标量场**（自旋 0），满足反对易关系。它的引入纯粹是技术性的，确保：
    1. 幺正性（物理态空间）
    2. 可重整性

### 鬼场的物理意义

- 鬼场只出现在虚拟过程（圈图）中
- 它们抵消规范场多余极点的贡献
- 保证 $S$-矩阵的幺正性

---

## 4.4 费曼规则

### QED 的费曼规则

| 元素 | 规则 |
|------|------|
| 光子传播子 | $\dfrac{-ig_{\mu\nu}}{k^2+i\epsilon}$ |
| 电子传播子 | $\dfrac{i(\slashed{k}+m)}{k^2-m^2+i\epsilon}$ |
| 顶点 | $-ig\gamma_\mu$ |
| 外电子线 | $u(p), \bar{u}(p)$ |
| 外光子线 | $\epsilon_\mu(k)$ |

### SQED（标量 QED）的额外顶点

- 标量-光子-光子顶点：$-ig(g_{\mu\nu} - ...)$

### QCD 的费曼规则

| 元素 | 规则 |
|------|------|
| 胶子传播子 | $\dfrac{-ig_{\mu\nu}}{k^2+i\epsilon}$ |
| 三胶子顶点 | $-gf^{abc}[...]$ |
| 四胶子顶点 | $-g^2[...] ... f^{abe}f^{cde} \pm ...$ |

---

## 4.5 Ward 等式

规范对称性导致 **Ward 等式**，它确保：

1. 规范不变性在量子层面得以保持
2. 重整化后耦合常数不减

形式上：

$$\partial_\mu \langle j^\mu(x) \phi(x_1) \cdots \phi(x_n) \rangle = 0$$
