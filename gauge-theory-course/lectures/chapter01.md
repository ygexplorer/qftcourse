# 第1章：基础回顾

> **前置知识**：初等量子场论（正则量子化、费曼图基础）

!!! abstract "本章概要"
    本章快速回顾量子场论的基本框架，为后续的路径积分量子化和规范场理论奠定基础。重点在于建立两种量子化方案的对比认知。

---

## 1.1 两种经典场论形式

### 拉格朗日形式

基于最小作用量原理，优点是保持明显的洛伦兹协变性：

$$\mathcal{L} = \mathcal{L}(\phi, \partial_\mu \phi)$$

作用量：

$$S = \int d^4x \, \mathcal{L}$$

### 哈密顿形式

基于正则动量 $\pi(x)$，破坏了明显的洛伦兹协变性，但物理图像更直接：

$$\mathcal{H} = \pi(x) \dot{\phi}(x) - \mathcal{L}$$

---

## 1.2 正则量子化

将经典场提升为算符，泊松括号提升为对易子：

$$\{ \phi(x), \pi(y) \} = \delta^3(x-y) \quad \to \quad [ \hat{\phi}(x), \hat{\pi}(y) ] = i\delta^3(x-y)$$

!!! warning "规范场的挑战"
    如果 $\pi_0 = 0$ 怎么办？（初级约束）正则量子化在处理规范场时会遇到巨大麻烦，需要使用 Dirac 括号。

---

## 1.3 LSZ 约化公式

LSZ 约化公式是联系格林函数与散射振幅的桥梁：

$$\langle 0 | T \phi(x_1) \cdots \phi(x_n) | 0 \rangle \xrightarrow{\text{on-shell}} \langle p_1 \cdots p_n | S | 0 \rangle$$

这是连接理论计算与实验观测的关键步骤。

---

## 1.4 施温格-戴森方程（SD方程）

量子场满足什么运动方程？让我们写出错误的经典方程提升式：

$$\square \phi + m^2 \phi = 0 \quad \text{（经典）} \quad \xrightarrow{?} \quad \text{（量子）}$$

### 算符形式推导

对于编时乘积，时间导数作用在阶跃函数上会产生 $\delta$ 函数，再结合等时对易关系产生**接触项**：

$$( \square_x + m^2 ) D(x-y) = -i\delta^4(x-y) + \text{contact terms}$$

!!! warning "推导的复杂性"
    算符推导非常繁琐，依赖非对易性和复杂的编时求导。这正是我们要学习路径积分的原因之一。
